功能描述
================================================================================
说明：
本样例主要展示USB模拟串口通信的功能，通过上位机设置串口通信参数(端口、波特率、数据位、校验位、停止位)，连接后通过上位机
串口调试助手发送数据给MCU，MCU接收到最后一个数据为0x0d时，将回发一串字符串给上位机。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）运行;
4）通过上位机将识别到两个COM口，通过上位机串口调试助手连接任意一个，向MCU发送数据，
所发送的数据以0x0d作为结束符，MCU接收到数据时将发送出如下字符串：
"This is a sample about usb cdc_1ep"或"This is a sample about usb cdc_3ep",
