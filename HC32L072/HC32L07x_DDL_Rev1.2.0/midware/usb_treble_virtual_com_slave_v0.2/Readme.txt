功能描述
================================================================================
说明：
    本样例主要展示USB模拟3路串口通信的功能，可通过上位机设置串口通信参数(端口、波特
率、数据位、校验位、停止位)，通过上位机打开3个串口调试助手，每个调试助手连接不同的
COM端口，通过调试助手发送数据给MCU，MCU接收到最后一个数据为0x0d的数据时，将回发一串
字符串给上位机的对应调试助手(发送数据的调试助手)。

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能；
3）运行;
4）通过上位机串口调试助手向MCU发送数据，所发送的数据以0x0d作为结束符，MCU接收到数
据时将发送出一串字符串给上位机,不同COM口MCU返回给PC的字符串不一样。

================================================================================
注意
不同的COM口接收到的字符串不一样。
