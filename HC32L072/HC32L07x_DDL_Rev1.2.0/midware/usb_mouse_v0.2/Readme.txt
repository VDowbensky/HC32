================================================================================
使用步骤
================================================================================
1）用USB线连接评估板与PC；
2)  打开工程并重新编译；
3）启动IDE的下载和调试功能；
4）运行;
5）
按下K1：光标向左移动，松开按键光标停止移动；
按下K3：光标向有移动，松开按键光标停止移动；
按下K2：光标向上移动，松开按键光标停止移动；
按下K4：光标向下移动，松开按键光标停止移动；
当光标停留在某个地方是，可以通过K5模拟鼠标的左键，如点击任务栏上的某个一打开的文件，当该文件最小化时，点击按键，该文件
将最大化，若该文件是已经最大化，则点击按键，该文件将最小化。

================================================================================
注意
若使用的是HC32LF07X-EVB-V11，则在文件board_evbhc32lf07x.h做一个宏定义：
#define BOARD_EVBHC32LF07X_V11
若使用的是HC32LF07X-EVB-V11，则取消以上的宏定义
